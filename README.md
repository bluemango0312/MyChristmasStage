## 🎧 About
**My Christmas Stage**는 크리스마스 무대를 주제로 한 **취향 월드컵 웹 프로젝트**입니다.
테스트를 하면서 다양한 연말 무대를 통해 크리스마스 분위기를 즐길 수 있었으면 좋겠습니다 :)
또한 온라인을 통해 취향을 공유하고 소통하는 문화가 발달되어 있는만큼, 공유 기능을 추가하여 사용자의 참여를 유도하였습니다.

약 3일간의 짧은 개발 기간으로 인해 디테일이 많이 떨어질 수 있지만 디자인부터 배포까지 새롭게 웹 개발을 해볼 수 있어서 재밌었던 프로젝트였습니다!

---

## ✨ Features
🎄 크리스마스 무대 취향 월드컵 </br>
🎶 선택 결과 기반 Top Picks 무대 구성</br>
📱 모바일 중심 반응형 UI</br>
🔗 X(트위터) · 카카오 공유 최적화</br>
🖼 Open Graph / Twitter Card 메타데이터 적용  </br>
